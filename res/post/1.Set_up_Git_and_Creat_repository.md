# Git安装与版本库创建
## Git安装
> Git安装后，需要进行下一步的设置。
```shell
$ git config --global user.name "YungJun"
$ git config --global user.email "caolianqun@gmail.com"
```
> 注意**git config**命令的 **--global**参数，表示这台机器上所有Git仓库都使用该设置。

## 创建版本库
1. 初始化仓库：`git init`

2. 添加文件到Git仓库，分两步：
   + `git add <file>`
    > 此命令可反复多次使用，添加多个文件；
   + `git commit -m <message>`
